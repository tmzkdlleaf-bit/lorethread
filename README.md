# 🌟 Lorethread — 로컬 실행 가이드

롤플레잉 특화 SNS 플랫폼입니다.

---

## 📋 시작 전 준비

### 1단계 — Node.js 설치
1. https://nodejs.org/ko 접속
2. **LTS** 버전 다운로드 (왼쪽 초록 버튼)
3. 설치 파일 실행 → 기본 설정으로 Next, Next, Install

설치 확인: 명령 프롬프트(cmd)에서 `node --version` 입력 시 버전 번호가 나오면 완료.

---

## 🚀 실행 방법

### 방법 A — 배치 파일로 실행 (가장 쉬움)
`시작하기.bat` 파일을 **더블클릭**하세요.

처음 실행 시 패키지 설치로 1~2분 걸립니다.
다음부터는 바로 시작됩니다.

### 방법 B — 명령 프롬프트로 실행
1. 이 폴더에서 주소창에 `cmd` 입력 후 Enter
2. 처음 한 번만: `npm install`
3. 매번: `node src/server.js`

---

## 🌐 접속
서버가 시작되면 브라우저에서:
```
http://localhost:3000
```

## ❓ 자주 묻는 질문

**Q: 데이터는 어디에 저장되나요?**
A: `db/lorethread.db` 파일에 모든 데이터가 저장됩니다. 이 파일을 백업하면 데이터를 보존할 수 있어요.

**Q: 다른 사람도 접속하게 하려면?**
A: 같은 와이파이 네트워크라면 본인 컴퓨터의 IP 주소(예: `192.168.0.5:3000`)로 접속 가능합니다.
인터넷 외부에서 접속하려면 Railway, Fly.io 등에 배포가 필요합니다.

**Q: 포트를 바꾸고 싶어요.**
A: `src/server.js` 파일에서 `const PORT = 3000;` 부분을 수정하세요.
